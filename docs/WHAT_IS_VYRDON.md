# What Is VYRDON

VYRDON is Root Routing Law. It governs proof, authority, transaction state, trace, risk, and execution before movement.

## Route Decision

```text
REQUEST
  │
  ▼
VYRDON ROUTE GATE
  │
  ├── proof
  ├── authority
  ├── state
  ├── trace
  └── risk
        │
        ▼
PASS / HOLD / BLOCK / CHALLENGE
        │
        ▼
EXECUTION OR STOP
```
