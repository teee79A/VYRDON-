# ما هي ڤيردون

ڤيردون قانون مسار جذري يحدد متى يمر القرار، ومتى يتوقف، ومتى يُمنع، قبل أن يتحرك المال أو الصلاحية أو التنفيذ.

## Route Decision

```text
REQUEST
  │
  ▼
VYRDON ROUTE GATE
  │
  ├── proof
  ├── authority
  ├── state
  ├── trace
  └── risk
        │
        ▼
PASS / HOLD / BLOCK / CHALLENGE
        │
        ▼
EXECUTION OR STOP
```
